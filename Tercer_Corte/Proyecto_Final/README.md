# 🏥 HealthSync: Gestión de Indicadores de Salud (ALSE 2026)

## 📋 Descripción
Aplicación de escritorio desarrollada en C++ y Qt5 diseñada para el monitoreo de indicadores biométricos. Integra SQLite3, visualización dinámica y cálculos metabólicos.

## ✨ Características Principales
* **Registro Biométrico:** Peso, presión arterial y glucosa con fecha y hora.
* **Análisis Estadístico:** Cálculo de promedios, tendencias e IMC.
* **Planificación Nutricional:** Cálculo de TMB (Mifflin-St Jeor) y metas calóricas.
* **Seguridad:** Autenticación de usuarios.
* **Exportación:** Generación de informes en CSV.

## 🛠️ Stack Tecnológico
* **Lenguaje:** C++ (OOP).
* **Framework:** Qt5 (Widgets & Charts).
* **Base de Datos:** SQLite3 (QSqlDatabase).
* **Documentación:** Doxygen.
