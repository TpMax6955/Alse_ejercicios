# 🏥 HealthSync: Gestión de Indicadores de Salud (ALSE 2026)

## 📋 Descripción
Aplicación de escritorio desarrollada en **C++** y **Qt5** diseñada para el monitoreo y registro de indicadores biométricos básicos. El proyecto integra persistencia de datos con **SQLite3**, visualización dinámica de tendencias y cálculos metabólicos avanzados.

## ✨ Características Principales
* [cite_start]**Registro Biométrico:** Seguimiento de peso, presión arterial y glucosa con marca de tiempo[cite: 31, 32].
* [cite_start]**Análisis Estadístico:** Cálculo de promedios, tendencias e Índice de Masa Corporal (IMC)[cite: 32, 63].
* [cite_start]**Planificación Nutricional:** Cálculo de TMB (Mifflin-St Jeor) y metas calóricas diarias[cite: 33, 49].
* [cite_start]**Seguridad:** Sistema de autenticación de usuarios para proteger datos sensibles[cite: 32, 53].
* [cite_start]**Visualización:** Gráficas dinámicas e interactivas integradas[cite: 63, 69].
* [cite_start]**Exportación:** Generación de informes en formato CSV[cite: 33, 59].
