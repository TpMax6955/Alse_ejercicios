Descripción

Este proyecto implementa la clase PanelSolar en C++ como parte del proyecto final del parcial.
La aplicación simula el registro de energía generada por paneles solares, permitiendo almacenar datos, calcular la energía total generada y obtener una eficiencia promedio.

Objetivos del proyecto

  • Aplicar programación orientada a objetos en C++
  • Implementar clases en archivos .h y .cpp
  • Usar contenedores de la STL
  • Organizar el proyecto con CMake
