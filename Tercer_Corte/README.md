# Alse_ejercicios

Este repositorio fue creado para la entrega de los trabajos, ejercicios y proyectos correspondientes al **segundo corte**.

Aquí se encuentran organizadas las diferentes actividades desarrolladas durante el curso, incluyendo ejercicios prácticos, entregas parciales y proyectos en **C++**, con el fin de llevar un control ordenado del trabajo realizado.

## Contenido del repositorio

Este repositorio está dividido en varias carpetas, cada una con un objetivo diferente:

- `Entrega_1/`: ejercicios y actividades correspondientes a la primera entrega.
- `Entrega_2/`: ejercicios y actividades correspondientes a la segunda entrega.
- `Entrega_3/`: ejercicios y actividades correspondientes a la tercera entrega.
- `Parcial/`: proyecto del parcial, desarrollado con una estructura más completa, incluyendo archivos fuente, compilación con CMake y documentación con Doxygen.

En estas carpetas se pueden encontrar:

- ejercicios de programación
- implementación de clases y objetos
- uso de estructuras y contenedores de la STL
- proyectos organizados en múltiples archivos
- documentación generada para algunas entregas.

## Tecnologías utilizadas

- **C++**
- **CMake**
- **Doxygen**
- **Git y GitHub**

## Observaciones

Algunas carpetas contienen ejercicios simples, mientras que otras incluyen proyectos más completos con estructura modular, documentación y archivos de configuración.
